# RefGuess
RefGuess single player game

Allows users to guess a random baseball reference page without knowing the given player. Uses BeautifulSoup4 for web scraping the player page with basic pandas operations for cleaning json/csv files associated with the player. Front end not live.

All game functions hand coded. Front end made partially by hand (I was learning some html/javascript skills through this project) but final product took my version and made it higher quality with Claude.
